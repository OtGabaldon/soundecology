    soundecology

    R package with functions to calculate indices for soundscape ecology and other ecology research that uses audio recordings.

    Indices

    • acoustic_complexity(): Acoustic Complexity Index (Pieretti, et al. 2011)
    • acoustic_diversity(): Acoustic Diversity Index (Villanueva-Rivera et al. 2011)
    • bioacoustic_index(): Bioacoustic Index (Boelman et al. 2007)
    • ndsi(): Normalized Difference Soundscape Index (REAL and Kasten et al. 2011)

    Calculate indices for many files

    The function multiple_sounds() lets the user calculate an index for all the wav files in a directory.

    Parallel processing

    The multiple_sounds() function has an optional argument, no_cores, that can be used to specify to run the function using several cores in the same computer. This argument uses the parallel package, standard in R since version 2.14.
